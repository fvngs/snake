
# snake

open source ultrasimplistic snake game



## features

- classic snake gameplay
- loads of customization options
- vs. AI mode


## installation

download python 3.10.* or above

clone the repository using git
```bash
  git clone https://github.com/fvngs/snake.git
```
run main.py
